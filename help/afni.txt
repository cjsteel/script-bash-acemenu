To install AFNI, run the following command:

tcsh @update.afni.binaries -package linux_ubuntu_16_64.tgz -do_extras

Note: Keep the '@update.afni.binaries' script 
in your home directory to run future updates!

For more info on how to install AFNI, refer to the official documentation:

https://afni.nimh.nih.gov/pub/dist/doc/htmldoc/background_install/install_instructs/steps_linux_ubuntu16.html
